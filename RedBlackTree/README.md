# 🟥⬛ 赤黒木 (RedBlackTree)

赤黒木の C 言語での実装例です。
`stdio.h` と `stdlib.h` のみ使用しています。

## コンパイル

```bash
gcc ./redblack.c -o redblack.out -g
```
